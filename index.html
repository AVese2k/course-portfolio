Workflow

My ideal workflow is that I would be able to connect the files from my computer to GitHub using Git. During this course, I copy and pasted the code from Khan Academy into a file in a GitHub repository, which is not the most effective workflow. In the future, I would like to learn how to use Git so I can effectively link my computer to GitHub, instead of just copy and pasting. Also, for big projects like a website, I could use the version control feature of GitHub to try various things in the code. By using this feature, i would be able to easily find a working version of my code if my experimenting did not work. Lastly, it would be easy to work with other people because you can fork their work and add your own code without ruining the other persons code. If the owner of the code likes what you did, they can merge your branch into their master branch.
